# 📘 Student Grade Tracker (Java Project)

## 📌 Project Overview
The **Student Grade Tracker** is a simple console-based Java application that allows users to input student names and marks, and then calculates important statistics such as average, highest, and lowest marks.

This project is built using **Object-Oriented Programming (OOP)** concepts in Java.

---

## 🎯 Features

- Add multiple students
- Store student name and marks
- Display all student records
- Calculate:
  - Average marks
  - Highest marks
  - Lowest marks
- Simple console-based interface

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class

---
